よくある質問

お部屋探しでよくいただくご質問にお答えします

Q 初期費用の目安はどのくらいですか？
A

一般的に家賃の4〜6ヶ月分が目安です。内訳は敷金（1〜2ヶ月）、礼金（0〜2ヶ月）、前家賃（1ヶ月）、仲介手数料（0.5〜1ヶ月）、火災保険料（約1〜2万円）となります。ホームナビでは「初期費用が安い物件」で絞り込み検索が可能です。

Q 申し込みから入居までの流れを教えてください
A

①物件を見つけて内見予約 → ②内見・比較検討 → ③入居申し込み → ④入居審査（3〜7日程度）→ ⑤重要事項説明・契約 → ⑥初期費用のお支払い → ⑦鍵の引き渡し・入居 という流れが一般的です。お急ぎの場合は即入居可の物件もございます。

Q 保証人がいない場合でも契約できますか？
A

はい、多くの物件で「保証会社利用可」となっています。保証会社を利用することで、連帯保証人なしでもご契約いただけます。費用は初回に月額賃料の50%〜100%程度、以降は年1万円程度の更新料が一般的です。

Q ペットを飼いたいのですが、対応物件はありますか？
A

ペット可・相談可の物件も多数掲載しております。「こだわり条件」から「ペット可」にチェックを入れて検索できます。ペットの種類・頭数によって条件が異なる場合がありますので、詳細はお問い合わせください。

Q 内見は無料ですか？
A

はい、内見は完全無料です。気になる物件がございましたら、お気軽にお問い合わせください。オンライン内見にも対応している物件もございますので、遠方の方も安心してご利用いただけます。

Q 家賃の値下げ交渉はできますか？
A

物件やオーナー様によりますが、交渉は可能な場合があります。特に空室期間が長い物件や、閑散期（6〜8月）は交渉に応じてもらいやすい傾向があります。ホームナビの担当スタッフが交渉のサポートもいたします。
