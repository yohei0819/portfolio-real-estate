# ホームナビ — 賃貸不動産ポートフォリオサイト

全国の賃貸マンション・アパート・一戸建てを検索・閲覧できるモダンな不動産ポータルサイトのポートフォリオです。  
フレームワーク不使用の Vanilla JS（ESM）で構築し、GSAP によるリッチなアニメーションとインタラクションを実装しています。

## ページ構成

| ページ | パス | 概要 |
|--------|------|------|
| トップ | `/` | ヒーロースライダー・新着物件・特徴セクション |
| 物件検索 | `/search.html` | フィルタリング付き検索結果一覧 |
| 物件詳細 | `/property.html?id=N` | 個別物件の詳細情報・初期費用シミュレーション |
| 駅・路線検索 | `/station.html` | 都道府県→路線→駅のステップ検索 |

## 主な機能

### 物件検索・フィルタリング
- エリア / 賃料 / 間取り / 物件タイプ / 面積 / 築年数 / こだわり条件で絞り込み
- 複数ソート（賃料昇降順・築年数・面積）
- ページネーション（10件/ページ）
- URL クエリパラメータによる状態管理（ブックマーク・共有対応）

### 駅・路線検索
- 都道府県→路線→駅の 3 ステップ選択 UI
- 物件件数付き駅一覧表示
- 選択状態のリアルタイム集計バー

### 物件詳細
- 基本情報 / 設備・特徴 / 間取り図 / 周辺施設マップ
- 初期費用シミュレーション自動計算
- 類似物件レコメンド

### UX・インタラクション
- GSAP ScrollTrigger によるスクロールアニメーション
- ページ遷移トランジション
- カーソルフォロワー（PC）
- ヒーロースライダー（自動再生 5 秒間隔）
- ローディング画面（プログレスバー）
- テキスト・ホバーアニメーション

### データ永続化（localStorage）
- お気に入り管理（ドロワー表示・他タブ同期）
- 閲覧履歴（最大 20 件）
- 検索履歴（最大 10 件・条件復元）

## アーキテクチャ

- **モジュールレジストリパターン**: `main.js` が現在ページを検出し、共通モジュール（静的 import）＋ ページ固有モジュール（動的 import）を初期化
- **イベント駆動**: `CustomEvent` によるモジュール間の疎結合通信
- **データファクトリ**: `PropertySeeds`（コンパクト定義）→ `PropertyFactory`（自動展開）→ `PropertyData`（完全オブジェクト）
- **純粋関数フィルタ**: `SearchFilter` が状態を持たず、URL パラメータから条件をパースしてフィルタリング

npm run build
